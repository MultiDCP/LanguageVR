# LanguageVR
이 레퍼지토리는 VR 환경 내에서 청각장애인들의 수화를 사용한 인터랙션을 연구하고자 하는 프로젝트입니다.

# Version
Unity 2021.3.9f1

# Requires
- HTC Vive Pro Eye + Vive Facial Tracker
- SteamVR & SteamVR Plugin
- HTC Unity Plugin
- XR Plugin Management
- Vive SRanipal Runtime
- Universal Render Pipeline
- Text Mesh Pro
